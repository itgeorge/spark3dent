# Bot Operating Procedures (BOPs)
These are instructions specific for this repo that should be executed by an agent, e.g. reviewing and updating licenses for used software.
They're intended to be used to automate tasks that aren't easy to automate with code, or to be used as shorthands before automating something with code.

## Format
Format should follow standard Markdown and should provide all necessary info for an agent to start work with:
- description of what the task is
- relevant files to look at
- specifics and gotchas of the task
- checklists if necessary
- (optionally) agent logs of previous BOP executions

## Updates
Monitor what the agent does well and where it gets stuck, and update the files accordingly. 
Agents can also modify the BOPs themselves to improve the next execution, or if required exlicitly by the BOP to log their work.
